JOGL Applet3 Graph Text Demo 01

Graph Text Demo Keys:

  • 1/2 - zoom in/out red-text
  • 6/7 - edit texture size if using VBAA (nope)
  • 0/9 - rotate the red-text
  • v - toggle vsync
  • i - live text input (CR ends it, backspace supported)
  • f - toggle fps
  • h - change text
  • space - change font
